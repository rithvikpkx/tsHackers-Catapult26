# Frontend (Builder B)

Next.js app shell for the Pulse Board, Tasks, Intervention, and Start Mode surfaces.

## Proposed pages

- `/` Pulse Board dashboard
- `/tasks` task list and risk tags
- `/why-stuck` explanation page
- `/urgent-update` intervention summary
- `/updated-schedule` before/after schedule
- `/start-mode` timer + next step panel

## Commands

```bash
npm install
npm run dev
```
