# Team Handoff (From Brief + Starter Tasks)

## Locked scope

- One believable student profile
- One ingestion source (seeded import is acceptable)
- One real intervention loop
- One clear before/after schedule action

## Builder ownership

- Builder A (you): backend ingestion, intervention endpoint, calendar action
- Builder B: Pulse Board + pages + polished UI
- Builder C: distortion + failure-risk outputs and explanation

## Integration checkpoints

- Hour 4: schema + route names frozen
- Hour 8: tasks load, dashboard shell up, corrected effort appears
- Hour 12: risk score + calendar action both working
- Hour 16: urgent update -> updated schedule -> start mode connected
- Hour 20: no new features, polish and reliability only
